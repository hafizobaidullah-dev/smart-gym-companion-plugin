# Smart Gym Companion Plugin

## 📌 Description
Smart Gym Companion is a WordPress plugin designed to assist users with fitness guidance, workout tracking, and gym-related management features.

---

## 🚀 Features
- Workout plan management
- User fitness tracking system
- Simple and interactive dashboard
- Gym routine organization
- Lightweight and fast performance

---

## 🛠️ Technologies Used
- PHP (WordPress Plugin Development)
- MySQL Database
- HTML, CSS, JavaScript
- WordPress Plugin API

---

## 🎯 Purpose
This project was developed to explore WordPress plugin development, backend logic, and user-centered fitness application design.
